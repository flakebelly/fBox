fComp

Multi-Mode Compressor

Nine Compression Engines - VST3 / AU

FETOptoVCAVari-MuPWNDiode BridgeDe-EsserZenerGate

fComp is a multi-mode mono/stereo compressor offering nine modes - FET, Opto, VCA, Vari-Mu, PWN, Diode Bridge, Zener, De-Esser, and Gate - each with analogue-modelled character and its own control set. The compressor always operates with analogue-style behaviour. There is no digital mode.

The Drive toggle adds additional harmonic saturation, transformer shaping, and noise floor that increase with how hard you drive the compressor. Every instance generates a unique serial number that seeds subtle hardware tolerances across the compressor, de-esser, and output stage. No two instances sound exactly alike.

Each mode presents its own set of controls because each models a different piece of hardware. FET mode gives you fast attack for aggressive peak control. Opto mode provides smooth, program-dependent levelling. VCA delivers transparent bus glue. Vari-Mu adds warm tube compression. Diode Bridge brings colourful vintage character with bias and recovery controls.

Three quality tiers - Eco, Studio, and Master - let you balance CPU against fidelity. Sidechain features include high-pass filter, external sidechain, stereo linking, and M/S processing.

Nine Modes

FET, Opto, VCA, Vari-Mu, PWN, Diode Bridge, De-Esser, Zener, and Gate - each a dedicated DSP engine, not a preset.

Drive Stage

Engage Drive for additional harmonic saturation, transformer shaping, and noise floor that responds to signal level.

Sidechain Tools

Sidechain HPF, external sidechain input, stereo link, sidechain listen, M/S processing, and I/O link for level-matched A/B.

Gain Match

One-click gain matching for honest A/B comparison. The plugin calculates makeup gain to match bypassed levels.
